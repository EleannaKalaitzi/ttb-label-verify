# Fixture label images

Synthetic distilled-spirits labels, one per compliance scenario. Generated by
`npm run fixtures` from [`src/lib/fixtures/specs.ts`](../src/lib/fixtures/specs.ts).
Each ships with its expected verdict — this is both the demo set and the
measurement corpus. Upload any of these in the app (mock mode needs no key).
